# VIGIL Signal Feed

VIGIL provides a high-fidelity, real-time signal feed for the Oil & Gas industry, delivering raw payload data, expert commodity tagging, and confidence scoring to streamline operational and trading decisions.

## Project Structure

- `backend/`: Node.js/TypeScript API
- `supabase/`: Supabase configuration and database migrations

## Getting Started

### Prerequisites

- Node.js
- Supabase CLI

### Setup

1. Clone the repository
2. Install dependencies in `backend/`:
   ```bash
   cd backend
   npm install
   ```
3. Initialize Supabase:
   ```bash
   supabase init
   ```
